$ usercfg

usercfg is an account management tool for developers.

Given a server implementing the specification, it allows for easy and passwordless command-line management of multiple accounts on multiple sites.

For more information, check out the GitHub project page.

INSTALL
curl https://pkamenarsky.github.io/usercfg/install.sh | /bin/bash
EXAMPLES
usercfg example.com create-user -n NAME -e EMAIL --ssh-key

Creates a new user. --ssh-key allows for the automatic upload of your public ssh key.

usercfg example.com update-user -n NAME -e NEW-EMAIL
usercfg example.com delete-user -n NAME
